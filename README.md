# My Bootstrap Project

This is a responsive Bootstrap redesign of Project 1 with 3 pages: Home, About, Contact.

## Features
- Responsive layout
- Bootstrap navbar and footer
- Hero sections
- Carousel
- Cards section
- Contact form

## How to Use
1. Clone the repository
2. Open `index.html` in your browser
3. Replace images in `/images` with your own
4. Customize content as needed
